# news_bot
Мой новостной бот для приложения вк
Чтобы бот начал работать нужно выполнить все пункты ниже:
1. Скачать файл bot.py с моего репозитория и поместить в папку с любым названием.
2. После этого нужно открыть программу Visual Studio Code и установить внутри неё Python. (подробнее можно посмотреть в этом вижео https://yandex.ru/video/preview/18332695999687147855)
3. Для установки библиотек нужно открыть терминал и ввести команду python -m venv (путь к вашей папке, в которой лежит файл bot.py)\venv #(обязательно нужно дописать в конце \venv), после этого наживаем ENTER.
4. После этого открываем новый терминал, если выдаёт ошибку, то ищем на ПК программу Windows PowerShell и запустить её от имени администратора и ввести Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser после этого нажать ENTER, на вопрос ответить Y и нажать ENTER, после этого заново открыть новый терминал в Visual Studio Code.
5. После это все библиотеки, что подчеркиваются жётой волнистой линией скачать(pip install имя библиотеки).
6. После этого открываем новый терминал и запускаем программу(в правом верхнем углу стрелочка).
7. После этого можно пользоваться ботом и писать ему сообщения.
