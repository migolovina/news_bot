# news_bot
Мой новостной бот для приложения ВК. Чтобы бот начал работать необходимо выполнить алгоритм, описанный ниже:

Алгоритм запуска новостного бота
1.	Скачать файл бота
1.1	Скачайте файл bot.py из моего репозитория.
1.2	Поместите его в любую новую папку (например, news_bot). 

2	Подготовить среду Visual Studio Code
2.1	Откройте программу Visual Studio Code.
2.2	Установите расширение Python внутри VS Code (если ещё не установлено).
2.3	При необходимости посмотрите видеоинструкцию: https://yandex.ru/video/preview/18332695999687147855)

3	Создать виртуальное окружение
3.1	Откройте терминал в VS Code (Терминал → Новый терминал).
3.2	Введите команду:
python -m venv путь_к_папке_с_bot.py\venv#
(например, python -m venv C:\Users\Имя\news_bot\venv)
3.3	Обязательно нужно дописать в конце \venv, после этого нажать ENTER.

4	Настроить выполнение сценариев PowerShell (если требуется)
4.1	Если при открытии нового терминала возникает ошибка, сделайте следующее:
4.1.1	Найдите в Windows программу PowerShell, запустите её от имени администратора.
4.1.2	Введите команду:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
4.1.3	Нажмите Enter, на вопрос ответьте Y (или Д), снова нажмите Enter.
4.2	После этого закройте PowerShell и перезапустите терминал в VS Code.

5	Установить недостающие библиотеки
5.1	В терминале VS Code (уже внутри виртуального окружения) выполните:
pip install имя_библиотеки
для каждой библиотеки, которая подчёркивается жёлтой волнистой линией в файле bot.py.

6	Запустить программу
6.1	Откройте новый терминал в VS Code (убедитесь, что виртуальное окружение активировано).
6.2	Нажмите на зелёную стрелочку в правом верхнем углу окна VS Code.

7	Пользоваться ботом
7.1	После успешного запуска бот начнёт работу.
7.2	Напишите ему сообщение в ВКонтакте – он должен ответить.
