# news_bot
Мой новостной бот для приложения ВК. Чтобы бот начал работать необходимо выполнить алгоритм, описанный ниже:

Алгоритм запуска новостного бота
1.	Скачать файл бота: скачайте файл bot.py из моего репозитория; поместите его в любую новую папку (например, news_bot). 
2.	Подготовить среду Visual Studio Code: откройте программу Visual Studio Code; установите расширение Python внутри VS Code (если ещё не установлено). При необходимости посмотрите видеоинструкцию: https://yandex.ru/video/preview/18332695999687147855)Создать виртуальное окружение: откройте терминал в VS Code (Терминал → Новый терминал); введите команду: python -m venv путь_к_папке_с_bot.py\venv (например, python -m venv C:\Users\Имя\news_bot\venv). Обязательно нужно дописать в конце \venv, после этого нажать ENTER.
3.	Настроить выполнение сценариев PowerShell (если требуется): если при открытии нового терминала возникает ошибка, сделайте следующее:найдите в Windows программу PowerShell, запустите её от имени администратора. Введите команду: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser. Нажмите Enter, на вопрос ответьте Y (или Д), снова нажмите Enter. После этого закройте PowerShell и перезапустите терминал в VS Code.
4.	Установить недостающие библиотеки: в терминале VS Code (уже внутри виртуального окружения) выполните: pip install имя_библиотеки (для каждой библиотеки, которая подчёркивается жёлтой волнистой линией в файле bot.py)
5.	Запустить программу: откройте новый терминал в VS Code (убедитесь, что виртуальное окружение активировано); нажмите на стрелочку в правом верхнем углу окна VS Code.
6.	Пользоваться ботом: после успешного запуска бот начнёт работу; напишите ему сообщение в ВКонтакте – он должен ответить.
